This repository documents my journey into the combined world of **space technology** and **embedded systems engineering**.

* Core concepts of **space science**, satellite systems, and mission technology
* Fundamentals of **embedded systems**, real-time computing, and firmware
* How embedded engineering powers spacecraft (OBC, avionics, communication, sensors)
* Daily exploration notes connecting **space + embedded**
* Short insights, summaries, and technical learnings

### 🚀 Why This Repository

To build a strong foundation for roles involving:

* Space systems engineering
* Avionics and flight computers
* Satellite embedded software
* Real-time mission electronics
